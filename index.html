¿Qué es y que función cumple el CSS?

CSS es un mecanismo complementario del lenguaje HTML que permite indicarle al navegador el estilo que debe darle a los distintos elementos al desplegar la información de un sitio web.

¿Cómo se incorpora el CSS dentro del HTML?

1- vinculando a un archivo de hoja de estilo separado 2- incrustando css en una etiqueta de estilo 3- agregando estilos en linea a elementos HTML 4- importando archivos de hojas de estilo existentes.

¿Cuál es la anatomía de una regla CSS?

p { color: red; width: 500px; border: 1px solid black; }

¿Cuáles son los tipos de selectores que puede utilizar CSS?

h1 { color: blue; } .special { color: blue; } h1, .special { color: blue; } h1, .special { color: blue; }
